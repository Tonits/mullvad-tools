# mullvad-tools

## About this project
This repository consists of two JS programs that can be of use when using Mullvad VPN in software.

## mullvad-relay-switcher
A JS program that switches to a random Mullvad-relay from a textfile list provided.

## mullvad-vpn-relay-list-parser
This JS program parses the vpn-list that Mullvad VPN prints in the console.
It produces a .txt file, which mullvad-relay-switcher uses.
